Acerca de este sitio

Para este sitio web, se utilizó:

  • HTML5 y CSS para la creación.
  • Bootstrap como framework frontend.
  • Enfoque "Mobile First" para la adaptabilidad.
  • Preprocesadores CSS para una organización eficiente.
  • Separación de presentación y contenido para futuras modificaciones en el diseño gráfico.
  • Metodología MaintainableCSS para un código CSS sostenible.
  • Estrategia de diseño centrada en la experiencia del usuario.
