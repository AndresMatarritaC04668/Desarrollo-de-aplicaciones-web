Mis Cursos Preferidos


Programación 1 y 2

Los cursos de Programación 1 y 2, impartidos por el profesor Edgar Casasola, se destacaron para mí. Mejoré mis habilidades en lógica de programación y dominé la sintaxis de lenguajes orientados a objetos como Java y C++. Estos cursos se enfocaron en el análisis de algoritmos y estructuras de datos de manera práctica.

Bases de datos

Se aprendió acerca del diseño e implementacion de bases de datos relacionales usando SQL Server.Además , se profundizó acerca de la normalización de las bases de datos para crear un diseño más óptimo y que se elimine la duplicación de datos innecesaria. Se profundizó en consultas avanzadas y tigres.

Paralela y concurrente

Se estudió el funcionamiento de los hilos de ejecución y la distribución de carga entre ellos al realizar tareas con mucho procesamiento e información. Para esto fue necesario el aprendizaje de las condiciones de carrera, técnicas para evitar dichas condiciones tales como : semáforos y mutex.Además se sistemas distribuidos.

Diseño de software

Un curso que se centró en los patrones de diseño del software. Se crearon programas y laboratorios resolviendo distintos problemas complejos usando patrones ya comprobados. También se abordaron aspectos como la creación de interfaces de usuario intuitivas y la planificación de la arquitectura de software escalable.
